# ✦ TaskFlow — To-Do List Web App

A modern, responsive, and feature-rich To-Do List application built using **HTML, CSS, and JavaScript**. TaskFlow helps users efficiently manage daily tasks with priority levels, progress tracking, and a clean UI.

---

## 🚀 Live Demo

[👉 *(Add your GitHub Pages link here later)*](https://amanmohanty18.github.io/taskflow-todo-app)

---

## 📌 Features

* ✅ Add, delete, and manage tasks
* 🎯 Task priority levels (High, Medium, Low)
* 📊 Real-time progress tracking bar
* 🔍 Filter tasks (All / Active / Completed)
* 🌙 Dark & Light mode toggle
* 💾 Data persistence using LocalStorage
* 🎨 Beautiful glassmorphism UI design
* ⚡ Smooth animations and transitions

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & UI (Glassmorphism, Animations)
* **JavaScript (Vanilla JS)** – Functionality & Logic

## ⚙️ How It Works

* Tasks are stored in the browser using **LocalStorage**
* Each task contains:

  * ID
  * Text
  * Completion status
  * Priority level
* Progress bar updates dynamically based on completed tasks
